The package includes the MATLAB code of the partial multi-view clustering algorithm PVC, which solves the multi-view clustering problems when views are partial.[1].

[1] S.-Y. Li, Y. Jiang nd Z.-H. Zhou. Partial Multi-View Clustering. In: Proceedings of the 28th AAAI Conference on Artificial Intelligence (AAAI'14),Qu¨¦bec, Canada ,2014.
 
For PVC, you can call the "PVCclust" function to run the clustering algorithm. 

You will find an example of using this code in the example directory. The example data is cornell. You can call function "main" to run it.
